# BookApp

一個讓你用寫網頁的方式，撰寫並出版 APP 的平台。

## 採用技術

在桌面端，我們使用 electron.js 處理。

在手機端我們還沒開始，預計使用 cordova.js 處理。

## 執行方法

請先安裝好 node.js, electron.js 之後，切到 electron 資料夾，然後執行本專案。

```
$ npm i -g electron
$ git clone https://github.com/ccckmit/bookApp.git
$ cd bookApp/electron
$ npm install
$ electron .
```
